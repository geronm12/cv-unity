# Geronimo Dev Room 🎮

Interactive retro-inspired portfolio built with Unity and designed as a playable experience instead of a traditional résumé.

The project represents my personal developer space — a pixel-art room where every object is interactive and connected to a part of my professional profile, projects, skills, and interests.

---

## ✨ Concept

Instead of presenting information through static cards or sections, this portfolio is designed as a small explorable game experience.

The room reflects:
- Software engineering
- Fullstack development
- AI enthusiasm
- Gaming culture
- Discipline and self-improvement
- Personal identity as a developer

Inspired by:
- Retro console menus
- Pixel-art adventure games
- Synthwave aesthetics
- Interactive indie experiences

---

## 🕹️ Planned Features

### Main Menu
- Retro animated UI
- Keyboard/gamepad navigation
- Language selection
- Download CV option
- Quick access links

### Interactive Room
Objects inside the room act as navigation points:

| Object | Purpose |
|---|---|
| 💻 PC Setup | Projects & architecture |
| 📚 Library | Skills & technologies |
| 🎮 Console / TV | Hobbies & gaming |
| 🥊 Gym Area | Discipline & mindset |
| 🖥️ Terminal | Technical experiments |
| 🖼️ Posters | Personality & inspirations |
